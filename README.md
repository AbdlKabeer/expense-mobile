# Expense Mobile

Expense Mobile is a lightweight and efficient expense tracking mobile application designed to help you stay on top of your finances.

## Features
- Quick and easy expense entry.
- Minimalistic UI for better user experience.
- Track expenses over time.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npx expo start
   ```
